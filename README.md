# Park Picker 

Welcome to Park Picker; a web application that allows users to search through National Parks by region, accessing detailed descriptions about our National Parks and the activities they offer. 

## Installation

Clone this repository and run

```$ npm i``` 

To install all necessary react packages and dependencies

## Usage 

This application is intended to display information and images from the National Parks api to users.
